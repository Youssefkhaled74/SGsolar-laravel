# Product Datasheets

This folder contains PDF datasheets for all solar water heater products.

## Current Files:
- `nobel-apollo-datasheet.pdf` - Nobel Apollo 200L
- `nobel-solface-datasheet.pdf` - Nobel Solface 150L
- `delpaso-solar-datasheet.pdf` - Delpaso Solar 300L
- `thermica-datasheet.pdf` - Thermica 250L
- `volca-datasheet.pdf` - Volca 100L
- `orca-heat-pipe-datasheet.pdf` - ORCA Heat Pipe System
- `evergreen-central-system-datasheet.pdf` - Evergreen Central System

## How It Works:
When users click the "Download PDF" button on any product card or in the product modal, the browser will automatically download the corresponding PDF file.

## To Update:
Simply replace the placeholder PDF files with your actual product datasheets, keeping the same filenames.
