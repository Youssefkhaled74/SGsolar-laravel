# Website Enhancement & Updates – Client Requirements

This document is a refined and professional version of the client’s requested edits and enhancements for the SG Solar website. It is organized by section to make implementation clear and structured.

---

## 1. Homepage

### Design & Layout

* Use the **second logo** as the primary logo on the homepage.
* Replace the static first section with a **slider**.

  * Each slide duration: **2 seconds**.
  * Slider should include **navigation arrows** (previous / next) so users can manually change images.
* Remove all text content from the first section.

  * This section should contain **images only**.

### Fixed Action Icons

* Add **fixed icons** visible while scrolling:

  * Call
  * WhatsApp
  * Contact Us

### About Us (Homepage Version)

* Use a **real photo** (not illustration or stock-style graphic).
* Ensure the **content matches exactly** with:

  * The About Us page
  * The provided Word document

### Products Section

* Display product **categories as cards**.
* Category card design should match the **product card design style**.
* Assign a **representative image** for each category.

### Navigation Behavior

* Navbar item order must match the **scroll order** of sections on the page.

### Projects → Portfolio

* Replace the **“Our Projects”** section with **“Our Portfolio”**.
* Portfolio should be divided by **project type**.

  * One section for each project type.

### Why Choose SG Solar

* Add a **“Why Choose SG Solar”** section.
* This section must appear **immediately after the About Us section**.

### Footer

* Add a **user feedback form** allowing visitors to submit feedback.
* Add a **News section** after the feedback form.

  * Display the **latest articles/news**.
* Add social media icons:

  * Facebook
  * Instagram
  * LinkedIn
  * TikTok
* Make the **“Etsal Bena”** title clickable and link it to the **Contact Us page**.
